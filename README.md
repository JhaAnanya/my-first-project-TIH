# House Sales Data Analysis

Hey there! This is my project where I dive into house sales data using Python. I wanted to see if I could find patterns in housing prices and figure out how different features affect the market.

## What I Did:
* **Cleaned the Data:** Used Python to handle missing values and get the dataset ready for analysis.
* **Made Predictions:** Built a Linear Regression model to see how accurately we can predict house prices.
* **Grouped the Data:** Used K-Means clustering to see how houses naturally group together based on their features.

## Tools Used:
* Python (Pandas, NumPy, Scikit-Learn)
* Jupyter Notebook / Google Colab
